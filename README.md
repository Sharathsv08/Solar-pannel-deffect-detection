# Solar-pannel-deffect-detection

The dataset comprises 200 images of Solar Panels taken using IR camera
which are annotated using Semantic Segmentation Annotation into 4
categories:  
a. ByPass Diode (BP)  
b. Dark Hot Spot (DHS)  
c. Light Hot Spot (LHS)  
d. Normal Panel
